# SCAN THEM ALL (GTSA) - Global Luxury Sweepstakes Platform

![GTSA Logo](https://img.shields.io/badge/Status-Production--Ready-brightgreen)
![Tech](https://img.shields.io/badge/Tech-HTML5%20%7C%20TailwindCSS%20%7C%20JS-blue)

GTSA is the world's first multi-category scan-based sweepstakes platform. Designed for high-conversion B2C lead generation and interactive consumer engagement, GTSA transforms everyday scanning into a luxury rewards experience.

## 🏆 Key Features

- **Billboard Game**: A rebranded, professional 5x5 interactive quest board where users scan local billboards to win entries. Includes real-time line detection and multi-round rewards.
- **$1,000 Cash Draw Funnel**: A strategic Two-Step Form Flow optimized for 2026 lead generation best practices.
  - **Step 1 (Hook)**: High-speed contact capture (Email/Phone).
  - **Step 2 (Segmentation)**: Demographic & interest data (Age, Gender, Category) for CRM-driven ad targeting.
- **Global Geo-Ecosystem**: IP-based country detection supporting 34+ territories with localized rules and advertiser categories.
- **Multi-Category Scanning**: Specialized tracks for Luxury Supercars, Dream Vacations, Gold Bullion, and Premium Accessories.
- **Professional Analytics**: Real-time position tracking and squad leaderboards for viral growth.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript, TailwindCSS (for high-performance utility styling).
- **Icons**: Lucide-Icons for premium, sleek visual language.
- **Animations**: Custom Framer-Motion style CSS animations for smooth transitions.
- **CRM Ready**: Pre-mapped logic for Copper CRM integration.

## 📂 Project Structure

- `index.html`: Main landing page and billboard game hub.
- `cash_draw.html`: High-conversion lead generation landing page.
- `js/bingo.js`: Core Billboard Game engine and behavioral data hooks.
- `js/lead-gen.js`: Multi-step form logic and segmentation system.
- `js/geo-rules.js`: Territorial compliance and dynamic local categories.

## ⚖️ Legal & Compliance

- Built with built-in disclaimers for "No Purchase Necessary" compliance.
- Supports regional exclusions (e.g., FL, NY, RI) and age-gating (18+).
